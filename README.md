## Requirements

Before starting üèÅ, you need to have [Git](https://git-scm.com) and [Node](https://nodejs.org/en/) installed.

### Starting

```bash
# Clone this project
$ git clone git@github.com:Thiago-spart/teste-pratico.git

# Access
$ cd teste-pratico

# Install dependencies
$ npm i

# Run the project in developer mode
$ npm run dev

# The server will initialize in <http://localhost:3000>
```
